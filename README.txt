TokenTube
=========


0. Description
==============

An integration component between LUKS/dm-crypt and PAM.


1. Building
===========

./configure
make
make install


2. Distribution integration
===========================

This is quite a bit of distribution-specifc work and you're on your own.
It's probably better to get an already customized source package from your
distribution.


3. Further documentation
========================

Refer to the documents in the subdirectory DOCUMENTATION.
